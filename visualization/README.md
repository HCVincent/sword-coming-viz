# 剑来可视化前端

React + TypeScript + Vite 前端，负责展示《剑来》统一知识库与编剧分析数据。

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 数据来源

前端默认读取 `public/data/` 下的文件：

- `book_config.json`
- `unit_progress_index.json`
- `unified_knowledge.json`
- `writer_insights.json`
- `swordcoming_book.json`
- `chapter_synopses.json`
- `key_events_index.json`
